* Meteor
* Node
* NPM

1. Clone the main Vulcan repo. (See )
1. Clone this repo, then `cd` into it
1. Run `npm install`
1. Rename `sample_settings.json` to `settings.json`
1. As per Vulcanâ€™s [Two-Repo Install](https://docs.vulcanjs.org/#Two-Repo-Install-Optional), launch V8 with:
```

```
